== Description ==

Bootstrap one page woocommerce checkout is an e-store extension that allows for faster checkout than ever before.
Bootstrap one page woocommerce checkout will allow users to complete the checkout process without spending time moving to a separate checkout page. 
After the user adds an item to their cart, the checkout fields and card of same page, allowing the user to complete the purchase process right then and there.
Advanced plugin settings allow to show your checkout page layout , you can show one Colum or two Colum for checkout fileds & card 


* You can choose one Colum or two Colum layout.
* You can choose to  manage your fields on your checkout.

== Installation ==

Install and Activate 

1. Unzip the downloaded 'Bootstrap-one-page-woocommerce-checkout' zip file
2. Upload the 'Bootstrap-one-page-woocommerce-checkout' folder and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate Bootstrap-one-page-woocommerce-checkout from Plugins page
4. 

Implement

1. Go to the wp admin Settings >> Checkout Settings
2. you can show one Colum or two Colum for checkout fileds & card
